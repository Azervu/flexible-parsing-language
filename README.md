

# Flexible Parsing Language


| Operator | Description                            |
|:---------|:---------------------------------------|
| `.`      | Operator Separator / Read              |
| `:`      | Write                                  |
| `{` `}`  | Branching                              |
| `(` `)`  | Grouping                               |
| `,`      | Parameter Separator                    |
| `*`      | Foreach                                |
| `~`      | Name                                   |
| `|`      | Transformer                            |
| `#`      | Lookup                                 |
| `##`     | Change Lookup Context                  |
| `'`      | Escape                                 |
| `"`      | Escape                                 |
| `\`      | Un Escape                              |
| `@`      | Context at start of current group      |
