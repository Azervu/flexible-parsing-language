

# Flexible Parsing Language


| Operator | Description                            |
|:---------|:---------------------------------------|
| `.`      | Operator Separator / Default Operator  |
| `:`      | Header                                 |
| `{` `}`  | Branching                              |
| `(` `)`  | Grouping                               |
| `*`      | Foreach                                |
| `~`      | Name                                   |
| `|`      | Transformer                            |
| `#`      | Lookup                                 |
| `##`     | Change Context                         |
